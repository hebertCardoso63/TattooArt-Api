# Tattoo Art

## Funcionalidades

- Autenticação
    * Registrar
    * Logar
- Tatuadores
  * Crud completo

## Como Iniciar o projeto

No diretório raiz do projeto utilize os seguintes comandos:

Para instalar as dependências do projeto:
  - `$npm install`

Para iniciar o projeto
  - `$npm start`
  - `$npm run start:dev`
